# leader-class

S.LSI Leader Class용 AI 하네스 실습 자료 저장소입니다.

현재 저장소는 별도 빌드 도구 없이 실행 가능한 단일 정적 HTML 문서 중심으로 구성되어 있습니다.

## 폴더 구조

```text
leader-class/
├── README.md
└── index.html
```

## 파일 설명

### `README.md`

저장소 개요와 현재 파일 구조를 설명하는 문서입니다.

### `index.html`

실습용 UI와 스타일, 인터랙션 로직이 모두 포함된 단일 페이지 파일입니다.

주요 구성은 다음과 같습니다.

- 프롬프트 설계
- 하네스 시뮬레이션
- 전략 브레인스토밍
- 프롬프트 워크플로우 안내

즉, HTML, CSS, JavaScript가 한 파일 안에 함께 들어 있는 구조입니다.

## 실행 방법

별도 설치 과정 없이 브라우저에서 `index.html` 파일을 열면 됩니다.

예시:

```bash
open index.html
```

macOS가 아닌 환경에서는 파일 브라우저에서 직접 열거나, 원하는 브라우저로 `index.html`을 실행하면 됩니다.

## 참고

- 현재 기준으로 루트 디렉터리에는 숨김 폴더 `.git`을 제외하고 `README.md`, `index.html` 두 파일만 있습니다.
- 향후 CSS, JavaScript, 이미지 자산이 분리되면 README의 구조 섹션도 함께 업데이트하는 것이 좋습니다.
